## MODIFIED Requirements

### Requirement: Contact Information Display
The contact page SHALL display the updated office address and working social media icons.

#### Scenario: Display new address
- **WHEN** the user views the contact page
- **THEN** the address shows "Sawangan Elok A 1 No 5 RT1 RW 10"
- **AND** a Google Maps link is available

#### Scenario: Click Google Maps link
- **WHEN** the user clicks the Google Maps link
- **THEN** it opens https://maps.app.goo.gl/8k6FPBdZhEkAh3Qq6

#### Scenario: Display social media icons
- **WHEN** the user views the contact page
- **THEN** social media icons (Facebook, Instagram, LinkedIn, Twitter, YouTube) are visible
- **AND** icons link to correct social media profiles

#### Scenario: Click social media icon
- **WHEN** the user clicks a social media icon
- **THEN** it navigates to the corresponding Esperion social media page
