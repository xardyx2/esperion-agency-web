## MODIFIED Requirements

### Requirement: Indonesian Locale Title Strategy
The Indonesian locale SHALL keep Tier 1 content (titles, headlines, navigation) in English per Translation Guidelines.

#### Scenario: Page titles in English
- **GIVEN** the user is viewing the Indonesian locale
- **WHEN** viewing page titles
- **THEN** titles remain in English (e.g., "Our Works", "Our Services", "Contact Us")

#### Scenario: Headlines in English
- **GIVEN** the user is viewing the Indonesian locale
- **WHEN** viewing section headlines
- **THEN** headlines remain in English (Tier 1 content strategy)

#### Scenario: Navigation labels in English
- **GIVEN** the user is viewing the Indonesian locale
- **WHEN** viewing navigation menu
- **THEN** navigation labels remain in English (e.g., "Home", "Our Works", "Our Services", "Articles", "About", "Contact Us")

#### Scenario: Descriptions in Indonesian
- **GIVEN** the user is viewing the Indonesian locale
- **WHEN** viewing descriptions and body text
- **THEN** descriptions are translated to Indonesian (Tier 2/3 content)
