## Instructions for Download

1. Kunjungi https://pixabay.com
2. Cari gambar sesuai dengan nama file di atas
3. Download dalam resolusi 1280px atau lebih
4. Simpan ke folder yang sesuai:
   - Gallery images: `frontend/public/works/`
   - Logo: `frontend/public/images/`

## Verification Checklist

- [ ] 48 gallery images terdownload (4 per work x 12 works)
- [ ] 1 logo image tersedia
- [ ] Semua gambar unik (tidak ada duplikat)
- [ ] Semua gambar relevan dengan konten proyek
- [ ] Tidak ada placeholder SVG yang masih dipakai
