# Product Context - Esperion Agency & Capital

## Project Overview

Esperion adalah platform digital agency yang terdiri dari dua dashboard utama:

1. **Dashboard Agency** (Phase 1 - Fokus Saat Ini)
   - Public website company profile dengan 6 halaman
   - Dashboard CMS untuk manajemen konten
   - Authentication system dengan JWT

2. **Dashboard Capital** (Phase 2 - Future)
   - TradingView webhook integration
   - Binance API integration untuk trading otomatis

## Business Goals

- Menyediakan kehadiran online profesional untuk Esperion Digital Agency
- Memungkinkan manajemen konten mandiri melalui dashboard CMS
- Menampilkan portfolio works dan services kepada calon klien
- Mengumpulkan leads melalui contact form dengan reCAPTCHA protection

## Target Users

1. **Visitors** - Mengunjungi public website untuk informasi services dan portfolio
2. **Admin/Editor** - Mengelola konten melalui dashboard CMS
3. **Viewer** - Read-only access ke dashboard
